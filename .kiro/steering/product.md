# Product Overview

Alaba+ is a multi-tenant church management system built with Django 6.0. It enables churches and religious organizations to manage their operations through isolated tenant schemas.

## Core Features

- Multi-tenant architecture with schema-based isolation per organization
- Member management (profiles, baptism records, status tracking)
- Ministry organization and member assignments
- Family relationship tracking
- Event planning with work groups and budget management
- Subscription management with Stripe integration
- Email-based authentication (no usernames)
- Background task processing for emails

## User Roles

- Owner: Full control of organization
- Admin: Administrative privileges
- Member: Standard access

## Target Users

Churches and religious organizations that need to manage members, ministries, families, and events in a secure, isolated environment.
